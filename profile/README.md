# Shared University Projects

This organization hosts a collection of university projects focused on AI-driven communication, distributed systems, and modern web development.

## Programme Information

These projects are developed as part of the **[N1SOF](https://www.gu.se/en/study-gothenburg/software-engineering-and-management-bachelors-programme-n1sof) (Software Engineering and Management Bachelor's Programme)** at the University of Gothenburg.

The programme is a 3-year (180 ECTS) international bachelor’s degree taught entirely in English, with a strong emphasis on both software engineering and software project management.

## Shared Technical Proficiencies
- Frontend: Next.js, Angular, Vue.js, Material UI
- Backend: Node.js, FastAPI, Express, TypeScript, Python
- AI/ML: Ollama, Tensorflow, PyTorch
- Infrastructure: Docker, Docker Compose, Kubernetes, Google Cloud
- Testing: Locust, Postman

## Development Workflow
- Issue Tracking: All tasks begin with a GitLab/GitHub issue.
- Branching: We use a [issue_number]-kebab-case naming convention.
- Standardized Commits: All commits follow an agreed uppon format to ensure a searchable history.
